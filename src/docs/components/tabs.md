## Default
```html
<div class="Tabs">
    <ul class="Tabs-nav">
        <li class="Tabs-nav-item is-active">
            <a data-tab="tab1">Tab 1</a>
        </li>
        <li class="Tabs-nav-item">
            <a data-tab="tab2">Tab 2</a>
        </li>
        <li class="Tabs-nav-item">
            <a data-tab="tab3">Tab 3</a>
        </li>
    </ul>
    <ul class="Tabs-content">
        <li class="Tabs-pane is-active" id="tab1">
            <p>Content for tab 1</p>
        </li>
        <li class="Tabs-pane" id="tab2">
            <p>Content for tab 2</p>
        </li>
        <li class="Tabs-pane" id="tab3">
            <p>Content for tab 3</p>
        </li>
    </ul>
</div>
```

## Class names

| Name                    | Default value | Description            |
|:------------------------|:--------------|:-----------------------|
| `$tabs`                 | Tabs          | Component name         |
| `$tabs-nav`             | Tabs-nav      | Navigation wrapper     |
| `$tabs-nav-item`        | Tabs-nav-item | Navigation wrapper     |
| `$tabs-nav-item-active` | is-active     | Active navigation item |
| `$tabs-content`         | Tabs-content  | Tabs content           |
| `$tabs-pane`            | Tabs-pane     | Tabs pane              |
