### Headings
```html
  <h1 class="Heading--xxlarge">Heading 1</h1>
  <h2 class="Heading--xlarge">Heading 2</h2>
  <h3 class="Heading--large">Heading 3</h3>
  <h4 class="Heading--medium">Heading 4</h4>
  <h5 class="Heading--small">Heading 5</h5>
  <h6 class="Heading--xsmall">Heading 6</h6>
```

### Body text
```html
<p>Body text</p>
```

#### Text tranform
```html
<p class="u-text-left">Body text aligned left</p>
<p class="u-text-center">Body text aligned center</p>
<p class="u-text-right">Body text aligned right</p>
```

### Lead text
```html
<p class="lead">Lead text</p>
```

### Lists
#### Unordered list
```html
<ul>
  <li>Morbi in sem quis dui placerat ornare. </li>
  <li>Praesent dapibus, neque id cursus faucibus, tortor neque egestas augue.</li>
  <li>Phasellus ultrices nulla quis nibh. Quisque a lectus. Donec consectetuer.</li>
  <li>Pellentesque fermentum dolor. Aliquam quam lectus, facilisis auctor, ultrices ut.</li>
</ul>
```
#### Ordered list
```html
<ol>
  <li>Morbi in sem quis dui placerat ornare. </li>

</ol>
```
